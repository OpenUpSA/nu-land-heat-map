Controlling The IFrame

To target an iframe give the element a name attribute e.g. 

<iframe
  ...
  name="map"
  ...
></iframe>

Now you can target that iframe using a with a target attribute e.g. 

<a 
  href="./"
  target="map">
  All Owner Types
</a>
<a
  href="./?filtered=City%20of%20Cape%20Town"
  target="map">
  City of Cape Town</a
>

e.g.

  • All Owner Types
  • City of Cape Town
  • Multi-Goverment
  • Multi-Goverment and National Government
